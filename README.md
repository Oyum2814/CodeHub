# CodeHub - Learn How to Code

CodeHub is your ultimate destination for unlocking the world of programming! Whether you're a seasoned developer looking to enhance your skills or a complete beginner eager to dive into the exciting realm of coding, CodeHub is your go-to resource.

This Web Application is a comprehensive learning platform that offers a wide array of coding tutorials, designed to cater to various skill levels and programming languages.

# Tech Stack

- **Frontend:**
  - [Next.js](https://nextjs.org/) - React framework for building the user interface.
  - [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework for styling.

- **Backend:**
  - [Node.js](https://nodejs.org/) - JavaScript runtime for server-side development.
  - [Prisma](https://www.prisma.io/) - Database toolkit for TypeScript and Node.js.
  - [MongoDB](https://www.mongodb.com/) - NoSQL database for storing application data.

- **Authentication:**
  - [NextAuth.js](https://next-auth.js.org/) - Authentication library for Next.js applications.

# Running Locally

## Prerequisites

Before running CodeHub locally, make sure you have the following prerequisites installed on your machine:

- [Node.js](https://nodejs.org/)
- [MongoDB](https://www.mongodb.com/)

## Clone the Repository

```bash
git clone https://github.com/your-username/CodeHub.git
cd CodeHub
```

## Install Dependencies
```bash
npm install 
```

## Configure Enviroment Variable
Create a .env file to store the following:

- DATABASE_URL 
- NEXT_AUTH_JWT_SECRET
- NEXT_AUTH_SECRET
- GITHUB_ID
- GITHUB_SECRET
- GOOGLE_CLIENT_ID
- GOOGLE_CLIENT_SECRET

## Run the Application
```bash
npm run dev
```
## Open Code in your Browser
 
 Visit http://localhost:3000 in your browser to access CodeHub
